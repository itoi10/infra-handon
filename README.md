serverless

chaliecはAWS LambdaでサーバレスアプリをPythonで構築するためのAmazon公式フレームワーク


- 新規作成  chalice new-project <name>
- デプロイ  chalice deploy --stage dev      
- 削除     chalice delete --stage dev 
- ローカル  chalice local --stage dev --port 8080
